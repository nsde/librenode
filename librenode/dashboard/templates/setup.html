FAQ

Why isn't there a "vanilla" option?

The answer is simple: there are just not a lot of reasons for it. Alternatives such as Paper and Purpur not only improve performance drastically, but also add new options and support for plugins.
Players don't need to install anything additionally to join servers which support plugins.

Why should I always use the latest Minecraft version?

One of the most important reasons is security. Older versions of Minecraft aren't supported and can lead to security vulnerabilities and other problems. Want the 1.8 PvP experience? Well, several mods and plugins can simulate exactly that. For example, using LabyMod can provide for more or less the same PvP experience as in 1.8.

Plugins & Mods - what's the difference?

Plugins don't need to be installed on the client (on the computers of the players) - but they have some drawbacks. For example, plugins can't really add new textures, sounds, or 3D models. Sometimes, workarounds are used, like player heads as mini-blocks or similar. Plugins can, in fact, add commands, gameplay changes and more.
You should use mods though when you want to play with completely new features, such as new monsters, blocks, items, and more.

How can I install & use datapacks?

Datapacks can be installed on all Minecraft servers with version 1.13 (17w43a) and above. Also, datapack support is a feature by Minecraft, not by its modded server software such as Fabric, Forge, Purpur, Paper and more.
Moreover, Datapacks don't need to be installed on clients (the computers of the players), just on the server.
