Tutorial

  1. Open up a Discord server you have high permissions or even admin on.
  2. Create a new text channel and call it something like #minecraft-log (you can change the name later).
  3. Right click the channel you've just created, choose Edit Channel and go to Integrations.
  4. Click the blue button saying Create Webhook. You can optionally choose an icon or name (like "LibreBot"). Make sure you've selected the correct channel.
  5. Finally, click Copy Webhook URL, paste the URL you've just copied in field below and you're done!
